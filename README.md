# Ecommerce-data-analysis
This repository shows the analysis of made-up ecommerce sales data from 14 sellers.

The Jupyter Notebook file (.ipynb) includes all Python codes for the project.
The Analysis Steps pdf file illustrates the step-by-step thinking process behind the analysis.
The Presentation pdf file summarizes the project and presents key findings.

The analysis includes mainly 2 parts: 
1. Exploratory analysis: https://public.tableau.com/app/profile/aiden.pham/viz/ChannelEngineAssignment/Performanceovertime?publish=yes
2. Demand forecastin (for 1 product category) using ARIMA
